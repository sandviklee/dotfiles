##### TPM

Installing TPM

```bash
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
```

Reloading Tmux

```bash
tmux source ~/.config/tmux/.tmux.conf
```

##### NVIM

Update TreeSitter (syntax tree generator)

```nvim
:TS Update
```

Update LazyVim

```nvim
:Lazy
```

#### The Magic

To sync these (using symlinks) with your local configs, use:

```bash
stow -t ~ .
```
